# 如何撤销一个应用在 iOS 上的精确位置访问

以下是如何阻止 iOS 应用程序访问您的 iPhone 精确位置。这样，它就能得到你所在位置的近似值。

无论你有一部[新 iPhone](http://xda-developers.com/best-iphone) ，一部[多功能 iPad](http://xda-developers.com/best-ipad) ，还是一部[大 Mac](http://xda-developers.com/best-macs)——苹果在它的操作系统中为你提供注重隐私的功能。这些允许用户分享他们选择的确切种类和数量的数据，并限制其余的数据。App 权限已经存在很多年了。它们允许您阻止应用程序访问特定的设备传感器或数据类型。权限在系统层面起作用，应用程序无法绕过它们。最近增加的一项隐私内容围绕着用户位置。它让你决定一个应用程序是获得你的大概位置还是精确位置。通过这种方式，应用程序无需了解您的确切坐标即可运行。这是本地新闻和天气应用的理想选择。您可以得到您想要的结果，而无需放弃您的实际位置。以下是如何在 iOS 上撤销应用程序的精确位置访问权限。

要撤销应用程序的精确位置访问并切换到 iOS 上的更广泛的估计，请执行以下步骤:

*   在你的 iPhone 上启动**设置**应用。

*   滚动到列表的最底部。
*   单击您想要限制的应用程序。

*   在底部，你会发现一个**精确定位**开关。

*   当你关闭它时，应用程序将不再能够告诉你的精确坐标。

请记住，不要取消 iOS 导航应用程序或地图上的精确位置访问是一个明智的想法。没有你的精确坐标，这些应用程序无法准确地指引你去任何地方，也无法在地图上显示你的实际位置。您可能还想为 iOS 消息应用程序保留精确位置。这样，当你和你的朋友分享你的位置时，他们会收到你的准确坐标，而不是他们的近似值。所以，在你撤销一个应用之前，一定要确定它是否需要你的精确位置。
