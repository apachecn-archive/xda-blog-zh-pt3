# 微软推出 Outlook for Windows 集成

> 原文：<https://www.xda-developers.com/microsoft-rolls-out-to-do-integration-outlook-windows/>

微软已经开始在 Windows 的 Outlook 应用程序中推出待办事项服务。这种新的集成允许您直接在 Outlook 应用程序中访问您的所有任务和待办事项，从而整合了更多的 Microsoft 365 生态系统。此更新现已面向当前渠道中运行 Outlook 版本 2207 的用户推出。

To Do 是微软的任务规划服务，在几年前微软收购 Wunderlist 后推出。它已经在其他一些地方与 Outlook 集成，特别是在网络上，你可以从 Outlook 的侧边栏访问待办事项。现在，它也是微软 365 附带的 Outlook 应用程序的一部分。

当然，Outlook 中的待办事项应用程序可以做你在其他平台上期望它做的任何事情。被标记的邮件会自动变成待办任务，所以你可以通过跳转到待办标签来轻松地跟踪你之前标记的邮件。如果您在组织中使用 Microsoft Planner，则在 Planner 中分配给您的任务也会显示在待办事项中。

再说一次，这并不是什么新功能，但它将所有待办事项功能都集成到了一个应用程序中，所以如果你习惯在 Outlook 中工作，现在使用起来会更容易一些。Outlook 中的待办事项应用程序看起来就像是 web 应用程序的包装器，所以这并不奇怪。您还可以获得一般提供的所有其他待办事项，包括突出显示当天到期任务的“我的一天”部分，以及您想要创建的所有待办事项列表。

如上所述，只要你有 Windows 版本 2207 (build 15427.20000)或更高版本的 Outlook，你就应该能够让新的微软进行集成，尽管它仍在推出。您也可以通过启用 Outlook 应用程序中的“即将推出”开关来获得 2201 版。这种新体验不是微软正在测试的新 Outlook 应用程序的一部分，它本身似乎是一个网页包装器，所以你仍然可以保留经典的 Outlook 体验，仍然可以访问待办事项。
