# TicWatch Pro 2020 是 Mobvoi 最新的 Wear OS 智能手表，配有 1GB RAM 和双显示屏

> 原文：<https://www.xda-developers.com/ticwatch-pro-2020-wear-os-1gb-ram/>

**更新(美国东部时间 3/9/20 @ 12:05PM):**TIC watch Pro 2020 现已开始购买，限时九折。

有几家公司仍然致力于生产 Wear OS 智能手表。其中一家公司是 Mobvoi，它发布了该平台提供的一些最好的设备。Mobvoi 在 2018 年发布了 TicWatch Pro [，提供了优质的规格和一些巧妙的技巧。去年，](https://www.xda-developers.com/ticwatch-pro-layered-display-30-day-battery/)[公司发布了 TicWatch Pro 的 LTE 版本](https://www.xda-developers.com/ticwatch-pro-4g-lte-first-impressions-two-displays-are-twice-as-nice/)。今天，Mobvoi 宣布其最新设备 TicWatch Pro 2020。

你可能记得 TicWatch Pro LTE 看起来与最初的 TicWatch Pro 基本相同。然而，除了 LTE 支持，它也有一些重要的升级。TicWatch Pro 2020 的升级与 LTE 型号相同，但这被认为是从 2018 年开始对最初只有 WiFi 的 TicWatch Pro 的更新。它不支持 LTE。

与原来的 TicWatch Pro 相比，最大的升级是 RAM。2020 型号的内存是原来的两倍，为 1GB。对于 Wear OS 设备来说，这实际上是一件大事。Wear OS 以缓慢著称，正如我在[对 LTE 模型的评论](https://www.xda-developers.com/ticwatch-pro-4g-lte-review-only-as-good-as-its-weakest-link/)中所述，额外的 RAM 带来了很大的不同。2020 年的型号也有 4GB 的内部存储，它由骁龙 Wear 2100 芯片组驱动([仍然没有 3100](https://www.xda-developers.com/qualcomm-snapdragon-wear-3100-sports-mode-still-coming/) )。

相对于原始 TicWatch Pro 的另一项升级是 MIL-STD-810G 耐用性评级。原版只有 IP68 认证，2020 型号也是如此。军用标准等级意味着它可以承受-22°F 到 158°F 之间的温度冲击，在-4°F 到 131°F 之间保持运行，57kpa 的压力，111°F 的太阳辐射，95%的湿度，盐雾，沙子，灰尘和冲击。

谈 TicWatch Pro 不能不提双层显示屏。Mobvoi 在最初的 TicWatch Pro 上引入了这一功能，2020 年的型号也有同样的出色功能。双层显示屏在 AMOLED 显示屏之上为您提供了额外的 LCD 显示屏。这个额外的显示器实际上充当了您的“永远在线”显示器，但它只需要很少的电力。如果你非常依赖额外的显示屏，你可以挤出长达 30 天的电池寿命。

从设计角度来看，最初的 TicWatch Pro 没有太大变化。其实我真的找不出设计有什么不同。LTE 型号在边框和按钮周围有滚花，但 2020 型号回到了最初的光滑表面。有两种颜色可供选择:黑色和银色。还有几种皮革和硅胶带可供选择。

TicWatch Pro 2020 将首先在英国推出，价格为 222.99 英镑。现在在英国可以从 [Mobvoi 的官方网站](https://www.mobvoi.com/us)和[亚马逊英国](https://www.amazon.co.uk/dp/B082Y2Q1LG?tag=xdadev04-20)购买。在美国，这款手表将于 3 月初在 Mobvoi 网站和亚马逊上以 259.99 美元的价格出售。
