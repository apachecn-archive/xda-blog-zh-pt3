# 现已上市:XDA-Android 用户开发者手册！[更新]

> 原文：<https://www.xda-developers.com/now-available-the-xda-developers-book-for-android-users/>

我们很自豪地宣布我们的第一本书的发布，XDA 开发者的 Android 黑客工具包。这本书由 Jason Tyler 撰写，并由 yours truly 投稿，它为新用户提供了他们所需要的东西，以便他们能够深入到 Android 修改的世界中。

这本书带你了解 Android 定制的基础，从各种示例设备开始。这些设备包括那些看似为修改而设计的设备，如谷歌 Nexus One，以及最近记忆中一些更加锁定的设备。这本书然后触及有趣的事情可以用你的新设备完成。最后，它包括设置和使用开始主题化和定制你自己的设备所需的工具，使它真正成为你自己的。总而言之，这本书旨在帮助新用户从零开始了解他们最喜欢的移动操作系统。

这本书的另一个好处是，它将帮助新 XDA 成员更好地利用论坛，因为它解释了网站是如何组织的，以及如何为你的手机或平板电脑找到和选择各种开发资产。

> 你渴望把你的安卓设备变成你自己的，但是你不知道从哪里开始？那么这本书是给你的。XDA 是世界上最受 Android 黑客爱好者欢迎的资源，一个巨大的社区已经围绕着用 XDA 定制 Android 设备而发展起来。 *XDA 的 android 黑客工具包*为你提供了通过入侵或扎根 Android 操作系统来定制你的设备所需的工具。
