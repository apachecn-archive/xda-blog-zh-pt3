# 如何为 iPhone 或 iPad 上的照片和视频创建 zip 文件

zip 使共享大量照片、视频或其他类型的文件变得更加容易。以下是如何在苹果 iPhone 或 iPad 上压缩几个文件。

如今，我们主动依赖智能手机拍摄日常生活的照片和视频。然而，与他人分享这些记忆有时会不方便，尤其是在同时分享大量媒体时。幸运的是，你可以在最近的 iPhones 手机上压缩照片和视频，允许你压缩这些文件，并将它们存储为一个 zip 文件，然后接收者可以解压缩。这个过程同样适用于[最新的 ipad](http://xda-developers.com/best-ipad)。因此，无论你是有一部 [iPhone 14 Pro Max](http://xda-developers.com/apple-iphone-14-pro-max-review) 还是不同的 iDevice，步骤都差不多。下面介绍如何在 iOS 和 iPadOS 上压缩照片和视频。

1.  在你的 iPhone 上启动**照片**应用。
2.  选择您想要添加到 zip 文件中的照片和/或视频。
3.  点击左下角的**分享**按钮。
4.  从共享菜单中点击**保存到文件**。
5.  选择目标文件夹，点击右上角的**保存**。
6.  在你的 iPhone 上启动**文件**应用。
7.  导航到保存照片/视频的文件夹。
8.  点击并按住它(触觉触摸)并选择**压缩**。
9.  瞧啊。您现在有一个 zip 文件，其中包含您选择的所有照片和视频。如果你喜欢的应用程序支持 zip 文件，你可以通过它们来共享 zip 文件，如果你更愿意与他人共享链接，你甚至可以将 zip 文件上传到云中。

* * *

如你所见，为 iPhone 或 iPad 上的照片和视频创建一个 zip 文件非常简单。如果您试图压缩其他文件类型，过程也是类似的。这不仅使组织和发送大文件批次变得更容易，而且在某些情况下，与单个未压缩文件相比，还可以减少这些文件占用的存储空间。
