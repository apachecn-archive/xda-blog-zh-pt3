# Google 文档、表单和幻灯片在 PC 上获得新的文档保存状态和离线指示器，在 Android 上获得新的评论 UI

谷歌文档、表单和幻灯片的最新功能包括保存状态、离线指示器和新的评论用户界面。

谷歌最成功的一些产品是它的办公应用程序、谷歌文档、表格和幻灯片。这些基于网络的微软办公软件的替代品因其协作工具、易用性以及免费的价格标签而非常受欢迎。谷歌一直在调整这些服务，最新的一批功能包括保存状态、离线指示器和新的评论用户界面。

谷歌正在更新桌面界面，该界面显示文档、表格和幻灯片的文件保存状态。保存状态现在直接显示在文档名称旁边，更具描述性。您可以轻松查看文档是保存到了云端还是设备上，因此不会混淆离线可用性。使文档脱机可用也更容易，因为您只需单击状态图标。这将在默认情况下启用，并在未来几周内对所有谷歌用户开放。

接下来是一个更加移动友好的谷歌文档、表格和幻灯片评论界面。相应的 Android 应用程序将获得一个更大、更清晰的界面，以便更容易看到评论及其上下文。有了新的快捷键和手势，可以很容易地快速滚动和回复评论。一个新的“@”按钮将允许用户快速提及某人。点击该按钮可以全屏打开评论线程。这个新的用户界面正在推出，所有谷歌用户都可以使用。

谷歌经常对文档、表格和幻灯片进行类似的小更新。这些小小的“生活质量”更新对确保谷歌的 office 应用套件保持领先地位大有帮助。
