OPPO F17系列和OPPO Enco W51 TWS耳塞在印度上市

原文：https://www.xda-developers.com/oppo-f17-pro-mid-range-smartphones-enco-w51-tws-earbuds-india/#0001-01-01

OPPO在2020年取得了很高的成功。随着中端Reno系列(形式为Reno 3和Reno 4)和Find系列(在 OPPO Find X2和X2 Pro )的继续，OPPO正在巩固其作为中端和高端细分市场顶级竞争者的地位。但实际上，该公司的低端智能手机产品在亚洲国家最受欢迎。今天，在印度发布了OPPO F17和F17 Pro以及OPPO Enco W51 TWS耳塞。

OPPO F17系列

正面看这些智能手机，它们的设计与其他OPPO智能手机有点偏离。背面有双哑光光泽饰面，与摄像头相交(想想Pixel智能手机过去的样子，但垂直排列而不是水平排列)，如果你发现Find X2令人兴奋，还有几个纯素皮革饰面。后置摄像头位于方形摄像头凸起中，类似于新款iPhones上的摄像头，只是我们有四个摄像头，而不是两个或三个。

OPPO F17是两者中较低端的，它拥有今年早些时候宣布的高通骁龙662 SoC。这是一款11纳米的芯片，配有八核CPU和高通的Adreno 610 GPU。这款SoC也为最近推出的摩托罗拉Moto G9提供动力，但到目前为止，我们还没有在其他智能手机上看到它。另一方面，OPPO F17 Pro采用了一款联发科Helio P95 ，这与印度版OPPO Reno3 Pro中的SoC相同。默认情况下，F17 Pro配有8GB内存和128GB内部存储，而F17的起始内存为4GB，内部存储为64GB。

这两款设备从后面看起来完全一样，但把它们转过来，你会发现不同之处。虽然F17 Pro在左对齐的打孔切口中有两个前置摄像头，但普通的F17有一个更传统的泪珠形切口，带有16MP摄像头；F17 Pro也有一个1600万像素的前置摄像头，但它配有一个辅助的200万像素深度传感器。后置摄像头也有所不同:F17 Pro有一个48MP的主传感器，尺寸为1/2”和f/1.8镜头，而F17有一个16MP的1/3.06”传感器和f/2.2镜头。然而，这就是分歧的终结。

两款手机都有6.4英寸AMOLED显示屏，分辨率为2400x1080，刷新率为60Hz，光学欠显示指纹扫描仪，OPPO的30W VOOC 4.0快速充电技术，通过USB 2.0 Type-C端口，4,000mAh电池，3.5毫米音频插孔，microSD卡插槽，以及顶部带有 ColorOS 7.2 的Android 10开箱即用。它们的背面都有相同的辅助摄像头，包括一个带有f/2.2镜头、1/4.0英寸大小和119˚ FoV的辅助800万像素超广角传感器，一个第三2MP微距摄像头和一个第四2MP深度传感器。

OPPO Enco W51

该公司还宣布了其最新的真正无线耳塞，也是第一款支持主动噪音消除(ANC)的OPPO耳塞:OPPO Enco W51。这些实际上内置了几个降噪功能，包括OPPO所说的“' 2+2 '降噪解决方案”，结合了双核降噪芯片和双ANC技术。耳塞支持无线充电，一次充电可以为您的耳塞充电长达24小时。

定价

OPPO F17 Pro将在印度的线下商店和₹22,990的线上商店零售，并将采用单8GB RAM + 128GB存储配置。该设备将有魔法蓝、哑光黑和金属白三种颜色可供选择。至于OPPO F17，我们还没有定价，但我们预计它会比Pro版本便宜很多。

OPPO Enco W51无线耳塞将在₹4,999零售，这对于真正具有主动降噪功能的无线耳塞来说是一个很好的价格。至于可用性，虽然，我们现在还不知道，但这些手机上市应该不会太久。
