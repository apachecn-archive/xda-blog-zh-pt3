# 这是下一代摩托罗拉·拉兹

> 原文：<https://www.xda-developers.com/motorola-razr-next-gen-leaked/>

早在一月份，我们独家报道了[我们认为是摩托罗拉 Razr 3](https://www.xda-developers.com/motorola-razr-3-snapdragon-8-gen-1-uwb/) 即将推出的规格，内部代号为“maven”。虽然我们已经知道在规格方面会有什么结果，但在设计方面我们并不确定。现在，由于来自 *91Mobiles* 的独家报道，我们第一次看到了这款设备的外部显示屏和内部可折叠显示屏。

根据该网站分享的图片，“maven”将摆脱其前身更怀旧的外观，失去下巴，取而代之的是更方正的设计。其带有打孔摄像头的 FHD+显示屏保持不变，而且似乎该手机将保留其比例较大的外部显示屏。看起来生活质量也有所提高，因为指纹识别器将从背面重新定位到电源按钮上。

至于它的相机设置，摩托罗拉 Razr 3 将有一个主相机，配有一个 50MP f/1.8 传感器，将与一个 13MP 传感器相结合，用于微距摄影。前置摄像头也以 1300 万像素的形式在显示屏上打孔。说到内部，它将由[高通的骁龙 8 代 1 系统芯片](https://www.xda-developers.com/qualcomm-snapdragon-8-gen-1/)驱动。该报告指出，骁龙 8 代 1 Plus (SM8475) SoC 也可以作为更高层次的选项提供，尽管该特定芯片组的生产延迟可能会导致这些计划无法实现。最后，将有 8GB RAM 或 12GB RAM 选项，内部存储采用 256GB 或 512GB 配置。

根据我们今年早些时候报道的规格，我们还预计内部显示屏将是 120Hz AMOLED 面板，手机将支持 UWB 和 NFC。DSCC 首席执行官罗斯·杨(Ross Young)也联系我们，告诉我们显示屏的尺寸是 6.7 英寸。

## ![Motorola Razr 3 inner display](img/f1a8c33ace7d7fe5c1f7d9693c9debad.png)

根据来自 91Mobiles 的报道，摩托罗拉 Razr 3 将有两种颜色:石英黑和宁静蓝。这款智能手机将于 7 月底或 8 月初抵达亚洲，随后将在全球发布。虽然该设备的价格未知，但从其规格来看，我们预计它会有一个与之匹配的旗舰价格。考虑到自 2020 年[摩托罗拉 Razr 首次推出以来](https://www.xda-developers.com/motorola-razr-5g-launch-date-in-the-u-s-unveiled-for-october-2/)[可折叠智能手机](https://www.xda-developers.com/best-foldable-phones/)市场已经变得相当拥挤，下一代手机会受到怎样的欢迎将是一件有趣的事情。
