荣誉20i设计

手机设计往往是第一印象的关键，Honor 20i就是一个很好的例子。Honor 20i的背面很容易让旁观者着迷，这主要归功于后壳上的渐变颜色。外壳光滑的表面可能会让人们误以为背面覆盖着玻璃，但事实并非如此，这有利于该设备的重量。我们收到的渐变蓝色从顶部的亮蓝色开始，向底部融合成亮紫色——这与我们在Redmi Note 7 Pro的类似颜色变体中看到的相反。

背面还装有垂直偏心放置的三后置摄像头和传统的指纹扫描仪。“HONOR”商标以及“AI CAMERA”文本标记非常突出，而其他监管标记在底部边缘非常模糊。该设备的底部有3.5毫米耳机插孔、麦克风孔、过时的microUSB端口和一组扬声器孔。设备顶部有辅助麦克风孔和混合SIM卡托盘。设备右侧有音量摇杆和电源键，左侧干净贫瘠。

请注意，荣誉20i是 而不是任何形式的防水 。这张封面图片仅仅证明了在一个热带气候的城市拍摄手机的难度。我们不建议以任何方式弄湿您的设备。

接下来，Honor 20i的正面被6.21英寸的LCD占据，它有一个水滴凹槽，一个薄的扬声器格栅，三个边缘上有中等大小的黑色挡板，底部边缘有一个稍大的挡板。与Redmi Note 7 Pro相比，挡板在所有边缘都更薄，屏幕与机身的比例更好，但Redmi Note 7 Pro的显示屏也略大。没有提到正面的大猩猩玻璃，所以我推测它不存在。手机自带预装的屏幕保护器，很难去掉(一点后我就不试了)。

就外观而言，Honor 20i击败了小米的竞争对手，这要归功于它的弧形背部设计，这使它比Redmi Note 7 Pro的四四方方的外观更具美学吸引力。Honor 20i的手感也要好得多，这要归功于弯曲的背部设计，重量(164克对红米Note 7 Pro的186克)给了我单手握持设备的信心。我已经痛苦地习惯了拿着更大更重的一加7 Pro ，所以我的手腕非常喜欢轻量级的Honor 20i。在我看来，玻璃背面的手机也被高估了(我是在玻璃背面的红米Note 7 Pro和一加7 Pro慢慢从我的桌子上滑下的时候说的)，我认为背面是光滑表面的聚碳酸酯外壳的实用性足以让我称赞这方面的荣誉。光滑的表面确实使Honor 20i成为指纹磁铁，我可以注意到光滑背面的微磨损，随着时间的推移，这可能可能复合。

在我看来，Honor 20i没有玻璃背板一点也不负面。毕竟，这款手机没有在高端市场竞争——这个任务留给了它的前辈们，即 Honor 20 和Honor 20 Pro——所以在这个价格范围内，我宁愿选择实用而不是高端。同样，这款手机也没有任何防水功能——当我在孟买恶劣的季风环境中艰难地在户外使用这款手机时，我非常想念这一点。考虑到设备的价格以及可靠的防水性在这个价格段仍然是一个白日梦的事实，我倾向于不要在这方面过于苛刻地判断手机。

对Honor 20i来说，一个绝对不利的因素是使用了microUSB而不是USB Type-C，尽管市场正在大力转向后者。与塑料外壳不同，microUSB端口确实令人遗憾地散发出削减成本的味道。除了在Android Go设备上，microUSB端口不再有实际意义，所以这里没有借口。

荣誉20i软件体验

Honor 20i运行在基于Android 9 Pie的EMUI 9上。我们的主编米莎尔·拉赫曼在一个独立的系列中对EMUI做了非常全面的评论[ EMUI 9评论第1部分 | EMUI 9评论第2部分 ]，所以我建议查看一下，以了解该操作系统的所有细微差别和更小的细节。

如果你是一个纯粹的安卓爱好者，你肯定会觉得被EMUI疏远了。但是，如果你曾经接触过大量定制的UX，比如小米的MIUI，像我一样接触过EMUI，这并不是一个很好的学习曲线。

默认情况下，启动器没有应用抽屉，因为EMUI会将所有内容放在主屏幕上。这在很大程度上是没问题的，但是EMUI没有提供基本的排序选项来让您的生活变得更轻松。结果是你的主屏幕变成了一大堆应用程序和图标，不同程度地难以找到你需要的应用程序，或者需要大量时间来以你觉得方便的方式整理你的主屏幕。幸运的是，你可以改变主屏幕样式来显示应用程序抽屉。更改主屏幕样式的设置可以在“设置”>“主屏幕与壁纸”>“主屏幕样式”中找到。默认启动器上的搜索栏非常漂亮，因为它将应用程序建议、联系人、日历和本地文件的结果组合在一起——所有这些都在一个地方，并根据它们的来源/类型整齐地分组。

我最不喜欢EMUI的是它激进的内存管理。Honor 20i配有4GB内存，我预计它足以在多个应用程序之间进行多任务处理。但这款设备的实际情况是，除了几个应用程序之外，我无法同时处理多个任务，否则就会丢掉工作。这是非常令人失望的，因为在这种情况下，操作系统似乎阻碍了硬件的发展。此外，Honor的UX的默认行为是优先考虑电池，而不是应用程序的连续性和体验，因此大多数应用程序会因为操作系统的行为而在后台关闭。这种默认行为给我带来了很多错过通知的问题，因为当设备显示屏关闭很长一段时间时，WhatsApp、Telegram和Discord(这些都是我经常需要的)等重要的即时消息应用程序通常会停止在后台同步。解锁设备后，我经常会收到大量通知。如果洪水没有发生，我需要记得单独打开应用程序，以确保实际上没有任何未决的通知，以防应用程序在后台被完全杀死。是的，这种行为是可以改变的，但是默认设置确实会给人留下糟糕的第一印象。我们将在完整的评论中更详细地探讨麒麟710的性能。

从负面来看，我喜欢EMUI的是它提供的电池备份，这要归功于前面提到的积极的内存管理。Honor 20i有一个相对“小”的3，400 mAh电池，但我仍然可以在正常情况下坚持大约4小时的屏幕时间和剩余20-30%的电池。我们将运行我们的标准电池基准，以确定该设备在持续大量使用的情况下有多好。其他较小的积极因素包括令人惊讶的可靠的面部解锁解决方案，可靠和快速的指纹解锁，以及总体愉快的体验(除了上面提到的例外)。EMUI还自带主题商店，用于定制应用图标、字体和锁屏风格。还有一个调频收音机应用程序，这是我的任何一加设备都没有的。

荣誉20i相机

Honor 20i的亮点是摄像头设置，这款设备在这方面不会令人失望。这款手机配备了一个三后置摄像头设置，包括一个24MP主传感器，一个8MP超宽传感器和一个2MP深度传感器。在正面，水滴凹槽内有一个32MP摄像头。该设备的USP因此是其在相机部门的多功能性，因为你有足够的装备来处理相当多的场景，比其他几个竞争对手可以声称做的更多。这种多功能性与价格的比率是Honor 20i的亮点，你不太可能在这一领域遇到其他令人信服的产品，可以声称是这款设备的最佳选择。

对于主相机的结果，当你有足够的光线时，你会得到好的清晰的照片。生成的图像在对比度、色彩准确度和饱和度方面达到了良好的平衡。这款手机的人工智能场景检测也令人惊讶地好，尽管它对图像的改变(如果有的话)对我来说几乎不明显。当光线开始变暗时，你会看到柔和和细节丢失的明显缺点悄悄出现。夜间模式设法让我拍出了相当清晰的照片，但这是目前为止唯一的亮点。

我喜欢Honor 20i的是它的广角镜头，它可以快速变成我最喜欢的辅助相机。广角确实增加了拍摄的背景，而不需要我们作为摄影师离得更远。Honor 20i的广角性能是可以接受的，但图像确实有一点红色。还可以通过广角镜头拍摄视频，这是某顶级旗舰上所缺少的。背面的第三个摄像头用于深度感应，通过相机应用程序中的光圈模式选项，你甚至可以对非人类物体应用背景模糊。然而，Honor 20i缺少某些功能，如4K视频录制和慢动作。

前置摄像头是32MP的射手，还过得去。前置摄像头试图模糊自拍的背景，但前置缺少第二个传感器似乎阻碍了它拍摄完美的散景照片。Honor 20i非常正确的做法是用这款相机进行面部解锁。我一拿起手机就解锁了，没有任何进一步的动作。相机没有被我的照片和我的弟弟愚弄(他和我有着惊人的相似，没有照片)。这种面部解锁实现的安全性与其他设备上的相同(不安全)，但这种设备上的便利性比我之前尝试过的其他设备要好。

第一印象结论

Honor 20i试图在印度市场拥挤的sub-₹15,000空间中竞争，该市场主要由小米及其Redmi Note系列主导，我不认为该设备有足够的力量将Redmi Note 7 Pro从其领先地位上赶下来。在这个领域还有其他高价值的设备，如令人印象深刻的Realme 3 Pro ，它提供了与Honor 20i上的麒麟710相当的性能主张，如果不是更好的话。
