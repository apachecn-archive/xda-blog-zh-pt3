华为手表3

标准华为Watch 3 采用圆形表盘，曲面玻璃屏幕，316L不锈钢外壳。它拥有1.43英寸的曲面AMOLED显示屏，峰值亮度为1000尼特，326ppi。这款手表有一个新的3D旋转表冠，华为表示，它可以感应不同程度的压力，并根据用户的输入提供触觉反馈。有四种不同的设计可供选择:Active(橡胶表带)、Classic(皮革表带)、Elite(金属表带)和Classic(米兰表带)。

Watch 3配备了一系列传感器，包括心率传感器和SpO2传感器、皮肤温度传感器、环境光传感器等。正如你对任何现代智能手表的期望，这款手表还可以跟踪你的睡眠和压力水平。这款手表配有2GB内存和16GB板载存储，但华为没有说明芯片组。

华为Watch 3的其他亮点包括支持4G的eSIM、100多种运动模式、内置GPS、5 ATM防水和无线充电支持。至于电池寿命，华为声称华为Watch 3在智能模式下可持续3天，在超长续航模式下可持续14天。

这款智能手表运行鸿蒙系统2.0，它还包括华为的AppGallery商店，允许用户直接在智能手表上安装鸿蒙系统应用。

华为Watch 3 Pro

华为Watch 3 Pro或多或少与标准型号相同，但具有更高级的设计和更长的电池寿命。它由钛制成，有蓝宝石玻璃保护显示屏。华为Watch 3 Pro在支持4G的情况下可持续使用5天，在省电模式下可持续使用21天。

 

定价和可用性

华为Watch 3将于6月11日在中国上市，起价为2600 CNY元(约合410美元)。与此同时，华为Watch 3 Pro的起价将为CNY 3300英镑(约合515美元)。两款手表今天都可以预购。目前还没有全球发布的消息。
