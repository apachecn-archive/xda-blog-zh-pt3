# Red Magic 3 的内置触发器如何改善您的游戏体验

XDA 电视台的 Max Weinbach 发布了一段视频，展示了红色魔法 3 上的触发器如何改善游戏体验。

我们最近谈论了很多关于游戏手机的话题，但是到底是什么让一部手机成为游戏手机呢？性能显然是能够玩密集型游戏的一大卖点。在手机上获得出色的游戏体验不仅仅是动力的问题。这就是为什么[努比亚红色魔法 3](https://www.xda-developers.com/red-magic-3-gaming-phone-north-america-europe/) 有内置触发器。

**[红色魔法 3 XDA 论坛](https://forum.xda-developers.com/red-magic-3)**

XDA 电视台的 Max Weinbach 发布了一段视频，展示了红色魔法 3 上的触发器如何改善游戏体验。这些触发器是电容性的和可定制的。Max 将触发器映射到[堡垒之夜移动](https://www.xda-developers.com/fortnite-android-bluetooth-controllers-60fps/)中的“scope in”和“fire”按钮。无论你把触发器放在哪里，它们都只是模仿屏幕点击，这意味着它们可以与任何有屏幕按钮的游戏一起工作。看看下面的视频，看看玩这个游戏有多简单。
