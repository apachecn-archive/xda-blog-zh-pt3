# Vaio 和 Gateway 在沃尔玛推出新的 Windows 11 笔记本电脑

> 原文：<https://www.xda-developers.com/gateway-vaio-windows-11-laptops-intel-12th-gen-walmart/>

Vaio 和 Gateway 都将从今天开始在沃尔玛推出新的 Windows 11 笔记本电脑。这些新的笔记本电脑配备了英特尔第 12 代处理器，并且有多种颜色选择和价格范围，因此您可以找到更适合您需求的产品。

从过去属于索尼的品牌 Vaio 开始，新的 Vaio FE 系列配备了 14.1 英寸全高清显示屏，并采用了第 12 代 U 系列处理器，特别是 15W TDP。这意味着您可以获得 10 个内核和 12 个线程，此外还可以拥有高达 16GB 的内存和 1TB 的固态硬盘。

这些笔记本电脑配有一个带内置隐私快门的全高清网络摄像头，一个用于 Windows Hello 的指纹读取器，以及一整套端口，包括 USB Type-C、USB Type-A、HDMI 和以太网。至于声音，它们还支持 THX 空间音频，这应该会提供更身临其境的体验。价格从 699 美元到 949 美元不等，取决于你的配置。

至于网关笔记本电脑，它们也配备了第 12 代处理器，最高可配英特尔酷睿 i7-1255U。它们还配备了最高支持触摸的全高清显示屏，以及最高 8GB 的内存和 512GB 的固态硬盘。这些也有几种颜色，但可用的选项取决于您选择的规格。

网关笔记本电脑的价格从 229 美元到 699 美元左右不等，这意味着规格将根据您选择的型号而有很大差异。最先进的型号似乎包括 Windows Hello 的指纹识别器和 200 万像素的网络摄像头，这使它们成为预算导向的设备。

如果你对新的 Gateway 笔记本电脑感兴趣，[你可以在沃尔玛这里查看它们](https://www.walmart.com/cp/8460364)。如果你更喜欢 Vaio 笔记本电脑，[你可以在这里找到它们](https://www.walmart.com/cp/4618390)。如果您想亲自购买，这些 Gateway 和 Vaio 笔记本电脑也将零售。
