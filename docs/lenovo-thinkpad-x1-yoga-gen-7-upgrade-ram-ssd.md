# 我可以升级联想 ThinkPad X1 Yoga Gen 7 中的 RAM 和 SSD 吗？

> 原文：<https://www.xda-developers.com/lenovo-thinkpad-x1-yoga-gen-7-upgrade-ram-ssd/>

联想 ThinkPad X1 Yoga Gen 7 是 2022 年你能买到的[最好的商务笔记本电脑](https://www.xda-developers.com/best-business-laptops/)之一。它建立在已经是商业用户的流行选择的基础上，但它在几个方面增加了赌注，包括具有 28W TDP 的新英特尔处理器、更快的 LPDDR5 RAM 等。不过，这在很大程度上仍然是一台商用笔记本电脑，客户通常希望从这些笔记本电脑中获得的一点是事后升级或更换组件的能力。如果你想升级联想 ThinkPad X1 Yoga 的 RAM 和/或 SSD 存储，我们有好消息和坏消息要告诉你。

虽然 ThinkPad X1 Yoga 的存储是以典型的 M.2 2280 SSD 的形式出现的，但 RAM 是焊接在主板上的。这意味着存储是可升级的，但 RAM 是不可升级的，您应该在结账时选择正确的容量。这与大多数[最好的笔记本电脑](https://www.xda-developers.com/best-laptops/)没有太大区别，考虑到这是为了让设备尽可能轻薄而做出的共同牺牲。然而，与更大的商用笔记本电脑相比，它可能有点令人失望。

## 你应该得到多少内存？

因为你以后不能升级它，所以你要确保你添加到联想 ThinkPad X1 Yoga 的 RAM 足以在你打算使用笔记本电脑的时候很好地为你服务。联想提供三种配置——8GB、16GB 或 32GB。其中，我们的一般建议是 16GB，但这取决于您的使用情况。

##### 联想 ThinkPad X1 Yoga Gen 7

联想 ThinkPad X1 Yoga Gen 7 是一款功能强大的可转换产品，采用第 12 代英特尔处理器和高达 32GB 的 LPDDR5 RAM，焊接在主板上。

我们会说 8GB 现在太少了。光是 Windows 11 就需要 4GB，如今只用 8GB 就能对多任务处理性能造成巨大打击。例如，现代浏览器使用大量的内存，所以如果你打开了大量的标签，你可能会注意到速度变慢。这也适用于打开各种不同的应用程序，以及从内存中加载大量数据的应用程序。其中包括数据库。

另一方面，32GB 可能超过了大多数人的需求。有了 16GB，绝大多数用户应该可以轻松地处理文档、浏览网页和做任何事情。拥有 32GB 显然更好，但这是一个昂贵的升级，所以我们只建议你知道你真的需要这么多。

## 如何升级联想 ThinkPad X1 Yoga 中的固态硬盘

至于存储，升级需要您可能期望的步骤。要进入 ThinkPad X1 Yoga，你需要一把飞利浦螺丝刀(一个 iFixit 修理包将包括你需要的所有工具，以及大多数其他笔记本电脑)，以及一些安全措施。首先，我们建议佩戴[防静电腕带](https://www.amazon.com/Wristband-Bracelet-Grounding-Alligator-Extendable/dp/B08CXQN86W?tag=xda-4g5pa52-20&ascsubtag=UUxdaUeUpU41191&asc_refurl=https%3A%2F%2Fwww.xda-developers.com%2Flenovo-thinkpad-x1-yoga-gen-7-upgrade-ram-ssd%2F&asc_campaign=Short-Term)以确保接地，同时避免在可能产生静电的布料或毛皮环境中工作。

除此之外，你还会想要新的固态硬盘，因为 ThinkPad X1 Yoga 支持 PCIe Gen 4，我们推荐像三星 980 Pro 这样的产品。它提供了一些你能得到的最快的速度，它是可靠的，所以你不能要求更多。

##### 三星 980 Pro

三星 980 Pro 固态硬盘支持 PCIe Gen 4，速度高达 7，000MB/s，是市场上最好的固态硬盘之一。

在这一点上值得注意的是，如果您更换 SSD，您将不会有任何数据当前在您的 PC 上。将你的数据备份到外部驱动器或 OneDrive 等云服务，然后在对你的电脑进行任何更改之前，按照我们关于[如何安装 Windows 11](https://www.xda-developers.com/download-windows-11/#DownloadISO) 的指南创建 USB 安装介质(这应该不同于你的备份驱动器)。

一旦你拥有了你需要的一切，除了确保笔记本电脑关闭并拔掉电源，你可能还想从禁用 ThinkPad X1 Yoga 内部的电池开始。这有助于防止电流通过机器，并避免您在笔记本电脑内工作时产生静电放电。要禁用电池，请按照下列步骤操作:

*   打开控制面板，将视图更改为**小图标**或**大图标**，然后点击**电源选项**。
*   在左侧菜单上，点击**选择电源按钮的功能**。
*   点击**更改当前不可用的设置**。您将需要管理员权限才能工作。
*   清除**打开快速启动(推荐)**复选框，然后点击**保存更改**。
*   重启电脑，看到联想 logo 时快速按 F1 进入 UEFI BIOS 设置。
*   导航到**配置**部分，选择**电源**，然后**禁用内置电池**。确认您的选择，笔记本电脑将关闭。

随着所有前期工作的完成，你终于可以更换固态硬盘了。你可以这样做:

*   将笔记本电脑倒置，让转轴背向您，用螺丝刀拧下固定底盖的五颗螺钉。
*   从顶部提起底盖，将其拆下。您可能需要使用撬动工具将其与机箱的其余部分分开。
*   固态硬盘靠近笔记本电脑的中间，就在电池的上方。它受到固态硬盘支架的保护，固态硬盘支架由两个螺钉固定。卸下两颗螺丝，提起 SSD 支架。
*   SSD 就露出来了，你可以把它滑出 M.2 插槽。
*   您还需要移除固态硬盘下方的散热垫。你应该能很容易地把它剥下来。
*   拿起新的 SSD，将连接器上的槽口与 SSD 插槽上的槽口对齐，然后以微小的角度将其滑入。SSD 的标签应该朝上。
*   将 SSD 支架放在 SSD 上，并将其与主板上的螺孔对齐。拧紧螺钉以将 SSD 固定到位，但注意不要拧得过紧。
*   按原样装上底盖，并用五颗螺丝将其固定回原位。同样，尽量避免过紧，因为过大的压力会损坏主板。

这就是关于升级联想 ThinkPad X1 Yoga Gen 7 的固态硬盘的所有信息。一旦您的电脑完好无损，您可以按照相反的顺序执行上述步骤，以启用电池和快速启动功能。您也可以参考我们的指南，使用我们上面提到的安装介质在您的新 SSD 上安装 Windows 11。
