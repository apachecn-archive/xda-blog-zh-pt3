# 如何在 Chromebook 上添加新的用户帐户

> 原文：<https://www.xda-developers.com/how-add-new-user-account-chromebook/>

如果你刚刚购买了一个新的 Chromebook 或 ChromeOS 平板电脑，你可能已经知道如何添加一个用户账户。但是你知道你可以添加更多吗？

向 Chromebook 添加新用户可以让你将不同个人的谷歌 Chrome 收藏夹、安卓应用和谷歌驱动文档放在不同的空间。与 Windows 和 macOS 需要登录现有帐户才能添加新帐户相比，谷歌使这个过程变得更容易，因为你可以在 ChromeOS 锁定屏幕上添加第二个帐户(或最多 19 个帐户)。你甚至可以不使用谷歌账户作为访客浏览网页。下面是如何开始的快速浏览。

## 如何向 Chromebook 添加新用户帐户

要向 Chromebook 添加新用户，你首先需要打开它的电源。Chromebook 启动后(应该只需要几秒钟)，不要像平常一样登录设备。相反，看着你的屏幕底部，按照下面的步骤。

1.  点击屏幕左下角显示的**添加人员**。
2.  选择谁在使用 Chromebook，要么**你**要么**一个孩子**。选择**一个孩子**将允许你设置额外的规则，哪些应用程序可以使用，哪些网站可以访问，哪些不能访问。
3.  点击**下一个**按钮。
4.  输入该用户的登录信息(Gmail 地址)。
    *   如果此人没有 Google 帐户，他们可以通过点击**创建帐户**并填写所需信息，在此页面注册一个权限。

5.  输入密码，然后按照屏幕上的步骤来个性化 Chromebook 体验。

添加这个新用户后，每次打开 Chromebook 时，你都会在锁定屏幕上看到两个用户。您可以通过单击配置文件图标并输入您配置的密码或 PIN 在两者之间切换。登录屏幕一次只会显示 19 个不同的用户。谷歌在一份支持文件中提到[如果你超过 19 个账户的限制，ChromeOS 可能会自动删除一个账户，作为一种节省空间的措施。](https://support.google.com/chromebook/answer/1059242#zippy=%2Cyour-account-was-deleted-from-your-chromebook)

## 如何在 Chromebook 上以访客身份浏览

如果你不想在 ChromeOS 上使用谷歌账户登录，你可以选择在 Chromebook 上以访客身份浏览。请记住，如果您以访客身份浏览，您的所有历史记录或文件都不会被保存，因此您必须离线保存文件，并单独记录您的历史记录。

1.  打开你的 Chromebook。
2.  在屏幕左下方，点击**作为访客浏览**选项。
3.  你的 Chromebook 会让你进入访客浏览会话。
4.  要退出客人浏览，点击日期和时间区域，并选择**退出客人**。

你会在 Chrome 浏览器上看到一个空白页，让你知道你正在以访客身份浏览。从这里开始，你现在可以在 Chromebook 上暂时不用谷歌账户浏览网页。你仍然可以保存文件和查看你的浏览历史，但一旦你退出，这些都将不复存在。确保通过云存储在线复制和保存文件，或者通过 USB 驱动器本地复制和保存文件。
