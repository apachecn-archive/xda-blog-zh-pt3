## 旗舰 Pixel 7 系列没有 microSD 扩展

Pixel 7 和 Pixel 7 Pro 都没有配备 microSD 卡插槽，这意味着你只能使用你购买的存储版本。不过，在 Pixel 7 系列上看到缺少 microSD 卡插槽并不奇怪。这是因为谷歌从未在其任何 Pixel 设备中添加 microSD 卡插槽。令人欣慰的是，基于云的服务和存储选项现在更加普及，这使得整个事情变得更容易接受。
