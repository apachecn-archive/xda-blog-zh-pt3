华为发布搭载英特尔酷睿i7-10510U处理器的新MateBook X和搭载AMD锐龙7 4800H处理器的14英寸MateBook

原文：https://www.xda-developers.com/huawei-announces-new-matebook-x-intel-core-i7-matebook-14-amd-ryzen-7-4800h/#0001-01-01

在2020年华为开发者大会上，华为将发布一系列公告，从智能手表和耳塞到高性能轻薄笔记本电脑。华为的笔记本电脑阵容基本包括三个系列:MateBook、 MateBook X/X Pro 和MateBook D。现在，中国OEM厂商正在这些系列下增加两款笔记本电脑，分别是新的MateBook X (2020)和MateBook 14 (2020)。

MateBook X (2020年)

MateBook X系列是高端轻薄笔记本电脑的故乡，专注于以时尚的外形提高最终用户的工作效率。

对于新的MateBook X (2020)，华为将赋予这款笔记本电脑一个新的无限全景显示屏，去掉了边缘的大量金属框架，为玻璃腾出更多空间。最终结果是，13.8英寸显示屏的屏幕与机身的比例达到90%。显示屏分辨率为3000 x 2000，278 ppi，完全100% SRGB覆盖。你还可以在这个触摸屏LTPS显示器上获得400尼特的峰值亮度。华为专注于使笔记本电脑轻薄便携，这就是为什么这款笔记本电脑比A4纸还要小，尺寸为284.4毫米x 206.7mm毫米x 13.6mm毫米，重量为1千克。

新款华为MateBook X (2020)上的全尺寸键盘配有1.3毫米的键程和背光。触摸板的尺寸也有所增加，总面积增加了26%。触摸板上的八个压电板负责触觉反馈。由于键盘区域实际上没有太多空间，扬声器位于Shift键下方。这款笔记本还在最靠近你的边缘配备了双麦克风，这对于通过视频通话进行通信来说应该很方便。在电源按钮上还有一个指纹传感器，在键盘上的按钮下有一个弹出式摄像头。这款无风扇笔记本电脑的冷却功能由导热铰链、蒸汽室散热模块和多层石墨层共同完成。

内部方面，华为将提供最高10代英特尔酷睿i7-10510U CPU和最高16GB RAM选项。这也是华为第一款配备Wi-Fi 6的笔记本电脑，理论峰值速度快3倍。板上有一块42瓦时的电池，应该可以持续9个小时播放FHD视频，足以在几天内完成大量以生产力为中心的办公室工作。就端口而言，你得到2个USB Type-C端口，任何一个都可以用来充电；和一个3.5毫米耳机插孔。

MateBook X (2020)还附带了一些漂亮的软件技巧。您可以获得内置的华为共享，从而在华为和带有EMUI 10和笔记本的Honor设备之间实现多屏协作功能。这款笔记本电脑将配备Windows 10家庭版。

价格和可用性信息尚不清楚，但MateBook X (2020)将提供银霜和森林绿色。

华为MateBook 14 (2020)

顾名思义，MateBook 14 (2020)是最新的14英寸触摸屏笔记本电脑，具有2160 x 1440分辨率的显示屏，100% sRGB覆盖率和90%的屏幕与机身比率。
