# Bing 聊天变得更加智能，有提示音、计数器和更吸引人的响应

> 原文：<https://www.xda-developers.com/bing-chat-gets-smarter/>

微软的人工智能必应聊天变得更加智能。该公司刚刚概述了针对聊天铃声、计数器、页面内容和聊天行为的四项体验变化，如果你是受邀试用的幸运儿之一，这些变化应该会改善你对预览体验的日常使用。

在所有四个变化中，最重要的是聊天语气。微软增加了一个新的开关，将 Bing chat 的人工智能语气从“更精确”改为其他选项，如“更有创意”或“更平衡”。为了让您知道自己处于哪种模式，每个模式都有不同的颜色。创意是紫色，平衡是蓝色，精确是绿色。微软表示,“精确”会让你得到更短的答案,“创造性”会产生更长更具描述性的答案,“平衡”是两者的混合。

其中一个更具争议的变化是转弯计数器。微软上周添加了这些内容，以防止 Bing chat 混淆，并且仍然将一次对话限制在六个响应。然而，现在你会在每个回答的底部看到一个计数器，让你知道对话的进展情况。当对话结束时，你会看到一个红色的停止灯，当它接近结束时，你会收到一个黄色的停止灯信号。

最后两个变化并不显著，但仍然值得一提。对于那些在 [Windows 11](https://www.xda-developers.com/windows-11/) 上使用新的微软 Edge Dev 的人来说，微软已经修复了 Bing 无法识别你正在浏览的页面上下文的问题。最后，聊天行为有一个调整，Bing 聊天不应该感觉不那么防御性和对抗性，而是更有吸引力。

本周有四个变化，但更多的变化很快就会到来。微软承诺每天都在改进这种体验，并将在微软必应博客上提供更新。
