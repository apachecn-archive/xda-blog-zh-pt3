# 华为 MatePad Pro 上的鸿蒙系统将 iPadOS 的 UI 带到了 Android 上

> 原文：<https://www.xda-developers.com/harmonyos-on-huawei-matepad-pro-impressions/>

上周，华为终于发布了新的移动设备操作系统。鸿蒙系统最早是在大约两年前宣布的，当时被认为是华为内部 Android 和 Linux 内核的替代者。华为推介鸿蒙系统的方式让它看起来像一个不可思议的工程壮举，因为新的操作系统和内核可能需要多年的开发才能涌现出来，但他们似乎不仅在几年内做到了这一点，而且还设法让它与 Android 二进制文件兼容。这就是问题所在:根据我们目前所看到的，智能手机和平板电脑的鸿蒙系统 2.0 似乎不是安卓的替代者，而是*是*安卓(基于)的。然而，即使是这样，我仍然喜欢鸿蒙系统在[新华为 MatePad Pro](https://www.xda-developers.com/huawei-matepad-pro-matepad-11-launch/) 上的表现，它的推出使华为处于自美国实施严重贸易限制以来的最佳位置。

## 华为 MatePad Pro 已接近商用

新款华为 MatePad Pro 的设计将其定位为笔记本电脑的替代品，其设计旨在提高工作效率。不仅显示屏很大(比去年的 MatePad Pro 大得多[，而且当设备处于横向模式时，摄像头也被移到了顶部挡板中。它非常适合 Zoom 通话和其他在线会议，特别是由于四个麦克风，华为表示，即使通过平板电脑的扬声器播放通话，也可以避免回声。](https://www.xda-developers.com/huawei-matepad-pro-review/)

这是我第一次发现自己也能在平板电脑上正确地进行多任务处理，这是我通常不会做的事情，因为我发现显示器太小了，不太可行。在最初的 mate pad Pro 上，尽管多任务是一个相当大的功能，但我从未享受过它。

八个扬声器发出响亮的声音，产生像样的低音，使平板电脑非常适合观看内容，尤其是其更轻的外形。可选的 M-Pencil 和磁性键盘增加了整体业务体验，使书写和绘图变得舒适和轻松。没有预装的应用程序可以使用铅笔(除了用于小涂鸦和其他内容的 Notes 应用程序)，尽管 AppGallery 上的 Canva 等应用程序可以原生使用 M-Pencil。

新的 MatePad Pro 还可以无线连接到华为笔记本电脑，用作辅助显示器，尽管我觉得它使用起来太慢太笨重。这不是最流畅的体验，尽管我可以看到它在紧要关头对在笔记本电脑的显示屏上工作时消费静态内容很有用。10，050 毫安时的电池应该可以让你轻松完成白天的任务，我已经用了几个小时了，甚至没有出汗。

YouTube 应用程序安装后就可以工作(尽管你不能登录，因为它依赖于 Google Play 服务)，Twitch 通过将你带到移动网站来工作。使用像 Twitch 这样的移动网站有一个缺点，因为它们通常缺乏本地应用程序所具备的功能。在 web 上使用 Twitch，只能使用自动分辨率调整。奇怪的是，推特只是....对我没用。花瓣搜索找到了一个 Twitter 的网页包装，AppGallery 本身把我指向了一个也不能用的移动网站。

如果我诚实的话，有一些应用程序在 MatePad Pro 上尝试和安装时完全是一团糟，Twitter 就是其中之一。我也希望 AppGallery 能让你选择安装这个应用，而不是把你指向那个网站，因为 Twitter 移动网站——虽然不是最差的——还不如实际应用本身成熟。华为对 AppGallery 的想法总体上是正确的，但它仍然迫切需要在一些领域进行一些调整和改进。遗憾的是，如果没有 Google Play 服务的设备使用了任何版权保护，那么你在谷歌 Play 商店上购买的任何应用都无法在该设备上使用。

不过这也不全是坏事，因为华为 MatePad Pro 有一些令人印象深刻的性能。Citra for Android 在《T2》的《精灵宝可梦 X》(pokémon X)中几乎完美运行，只有一些小故障。麒麟芯片组在过去通常在仿真性能方面落后，所以很高兴看到这款设备在 Citra 中表现很好。

## 鸿蒙系统 2.0 很可能是基于安卓系统的，但这并没有错

华为几乎没有反驳针对移动设备的鸿蒙系统 2.0 是 Android 的一个分支的指控。另一方面，[华为 Watch 3 系列](https://www.xda-developers.com/huawei-watch-3-pro/)的鸿蒙系统 2.0 显然不是基于 Android 而是 LiteOS，但华为的智能手表已经多年没有运行 Android 了。当被问及鸿蒙系统时，华为表现得吞吞吐吐令人失望，但作为安卓的一个分支并不会让这个操作系统本身令人失望。

鸿蒙系统 2.0 全面引入了许多 UI 改进，尤其是在最右边的快速设置菜单中。我不会说谎——这里有一些*非常明显的* iPadOS 的影响，有时很难忽略它们。甚至系统图标看起来也很有苹果风格。

AppGallery 自己的设计还有很多需要改进的地方，大图标占据了主页的大部分显示。并不是说它很杂乱，而是感觉像是浪费了空间。多任务菜单也比较缺乏想象力，向上滑动就能刷走应用。这里实际上与 EMUI 没有太多不同，这要么意味着华为在移植熟悉的 UI 方面做得很好，要么也有可能这只是 EMUI 的下一个版本。

我们之前已经暗示过它，但是如果你想知道为什么我们对鸿蒙系统 2.0 *到底是什么*感到困惑，那是因为我们看到的或得到的所有声明都没有真正澄清事情。例如， [*TechCrunch*](https://techcrunch.com/2021/06/02/huawei-officially-launches-android-alternative-harmonyos-for-smartphones/) 报道称，“华为否认了鸿蒙系统是安卓操作系统衍生产品的猜测，并表示没有一行代码与安卓的完全相同。”该出版物还报道称，“华为的一名发言人拒绝透露该操作系统是否基于 Linux，即驱动 Android 的内核。”如果是这样的话，那么它是如何工作的，Android 应用程序是如何兼容的？在给德国网站 [*ComputerBase*](https://www.computerbase.de/2021-06/updates-fuer-diese-geraete-harmonyos-loest-android-aber-nicht-ganz-das-aosp-ab/) 的一份声明中，华为消费者业务集团软件部总裁王博士表示，“华为在遵守开源许可规则并履行相关责任和义务的情况下，使用来自的开源代码。”他说，这是为了“确保我们现有的用户仍然可以享受他们在我们的手机和平板电脑上熟悉的体验。”在阅读了这些陈述后，我们*仍然*不确定鸿蒙系统 2.0 和 Android 之间的确切关系，所以我们联系了华为，并收到了以下陈述:

“为了保护现有手机和平板电脑用户的数字资产，鸿蒙系统 2 目前允许现有的 Android 应用程序在鸿蒙系统 2 的一些设备上运行，华为遵守了相关的开源许可规则。集成了 HMS Core 的 Android 应用程序可以继续在鸿蒙系统上运行。鸿蒙系统 2 是华为基于开源项目 OpenHarmony 2.0 为不同场景下使用的智能设备开发的商业版本，继承了华为在 EMUI 上的差异化能力和专有技术。Android 是基于 AOSP 开源项目开发的，而 OpenHarmony 是由 OpenAtom 基金会创建和运营的开源项目。作为其主要贡献者之一，华为希望全球更多的厂商和开发者加入到 OpenHarmony 项目中来。这样，我们都可以为开源社区的发展做出贡献，从而改善消费者的体验。”——华为发言人

我们将继续跟进并深入研究鸿蒙系统 2.0，以了解更多相关信息。但即使是另一个 Android 分支，我也不认为这有什么问题。从零开始制作一个操作系统并非易事，鉴于 Android 的开源特性和 EMUI 的多年工作，华为希望在此基础上进行开发是有道理的。

和制造，因为新的 MatePad Pro 是我自贸易禁令以来使用的最好的华为设备。该公司终于在 HMS Core 和 AppGallery 上找到了立足点。这是重塑品牌的好时机吗？有可能，但重要的是，华为要对客户更加透明，让他们知道鸿蒙系统 2.0 是什么，不是什么。

## 结论

新的 MatePad Pro 对华为来说是一款有趣的设备，因为我们可以在这里看到新软件身份的开端。鸿蒙系统和华为 AppGallery 的现状是该公司努力克服美国贸易禁令的高潮。虽然鸿蒙系统 2.0 并不符合我们的承诺，但它是华为重振其软件身份的必要版本。

远离安卓品牌有助于它识别其他东西，因为“安卓”在更广泛的世界中被认为*需要*谷歌。如果不是安卓，那它为什么需要谷歌？鸿蒙系统可以被视为一个不同的或独特的操作系统，支持所有你喜欢的应用程序，即使它是引擎盖下的 Android。任何像你我一样了解并热爱技术的人都能看穿这种错觉，但这不是华为在这里的主要市场。完全有可能，新的公众认知正是华为成功所需要的，远离被谷歌“禁止”的耻辱。现在不用谷歌是一种选择。这是华为在走自己的路。从技术层面上来说，它是“安卓”，但它不是人们所期待的谷歌意义上的“安卓”。

不过，华为 MatePad Pro 还没有关于定价或可用性的信息。
