# 微软正在测试一款重新设计的漂亮的 Windows 11 照片应用

> 原文：<https://www.xda-developers.com/new-windows-11-photos-app-in-testing/>

微软正在测试 Windows 11 中的新照片应用。新的应用程序可供开发频道的 Windows Insider 测试人员使用，并带来了完整的重新设计和一些非常受欢迎的功能。它通过对版本 2022.31090.16004.0 的更新，取代了 Windows 开发频道版本上现有的照片应用程序。

新的 Windows 11 照片应用程序的亮点是图库视图。与当前版本的 Windows 11 相比——它是基于 Windows 10 的照片应用程序设计的——这种新体验使查找、管理和浏览照片变得更容易。它还采用了 Windows 11 的许多设计元素，如圆角、侧边栏和新的云母效果。此外，就像 iOS、Android 和网络上的 OneDrive 一样，甚至还有一个“回忆”选项，可以带你回到一年前或过去的照片。微软甚至增加了将照片轻松备份到 OneDrive 的功能。现在，侧边栏中有一个可视指示器，显示 OneDrive 存储的使用量，每张照片上有一个云图标，显示照片已发送或在 OneDrive 云中。

这款照片应用的其他较小的新功能包括新的多窗口和多屏视图。微软甚至让你更容易通过新的侧边栏直接从照片应用程序导入和管理相机和手机中的照片。

尽管增加了所有这些功能，但微软确实从这个新应用中删除了一个东西。随着 Clipchamp 成为 Windows 11 中新的视频编辑器，新的照片应用程序删除了使用传统视频编辑器创建和编辑快速视频的功能。不过不要惊慌，因为微软正在将传统照片应用作为 [Microsoft Photos Legacy](https://www.microsoft.com/store/apps/9NV2L4XVMCXM) 在微软商店出售。
