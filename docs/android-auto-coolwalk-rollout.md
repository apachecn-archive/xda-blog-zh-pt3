# Android Auto 的分屏重新设计终于开始推出

> 原文：<https://www.xda-developers.com/android-auto-coolwalk-rollout/>

谷歌[在去年的 I/O 开发者大会上展示了一个更新的 Android Auto UI](https://www.xda-developers.com/android-auto-google-io-redesign/) ，代号为 Coolwalk。它采用了新的分屏布局，能够同时显示导航、音乐播放控制等。虽然谷歌表示重新设计将在“今年夏天”向用户推出，但它并没有随着八月份的 [Android Auto 8.0](https://www.xda-developers.com/android-auto-8-0-rollout/) 更新一起到来。该公司在 11 月推出了测试版，现在终于覆盖了所有用户。

谷歌在正在进行的 CES 2023 贸易展上宣布推出，以及几个面向 Android 汽车用户的新助手功能。在一篇强调谷歌所有汽车相关公告的博客文章中，该公司指出，更新后的 Android Auto UI 优先考虑了司机的三个重要目标:*“导航你要去的地方，与朋友和家人交流，播放你的音乐或播客。”*

重新设计使地图用户界面更接近驾驶员的座位，使导航更容易一目了然，增加了一个新的媒体卡，用于快速访问播放控制，以及一个快速启动，让你打开最近使用的应用程序。这种 UI 可以自动扩展不同大小的 Android Auto head 单元，无论用户驾驶哪辆车，都可以提供出色的体验。

谷歌还向 Android Auto 推出了新的助手支持的智能建议，以帮助司机回复未接来电或开始播放建议的媒体。此外，此次更新为媒体带来了一个新的可查看的进度条，以改善播放控制和 WhatsApp 呼叫支持最新的三星和 Pixel 智能手机。

随着 Android Auto 的变化，谷歌还透露，对数字汽车钥匙的钥匙共享支持将很快进入三星和小米设备，并扩展到更多汽车品牌。最后，该公司宣布支持 Android Automotive 的高清地图，该地图将显示车道标志、标志和路障等细节。
