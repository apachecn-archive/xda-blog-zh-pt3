# 如何从 Windows 11 家庭版升级到专业版

> 原文：<https://www.xda-developers.com/upgrade-windows-11-home-pro/>

[Windows 11](https://www.xda-developers.com/windows-11/) 现已正式发布，就像之前的 Windows 版本一样，它将有几个不同的版本。Windows 11 版本与 Windows 10 相似，不同的版本包含某些不同的功能，这些功能使操作系统更适合不同的用户。对于普通大众来说，你会发现有两个主要版本——Windows 11 家庭版和 Windows 11 专业版。顾名思义，Windows 11 Pro 是功能更丰富的版本，但大多数笔记本电脑都配有家庭版。即使是微软自己的 [Surface 设备](https://www.xda-developers.com/best-microsoft-surface-pcs/)也配有 Windows 10 Home，除非你购买它们的商业版本，这些版本配有 Windows 11 Pro。在本指南中，我们将向您展示如何从 Windows 11 家庭版升级到专业版，以及为什么您可能想要这样做。

如果你想了解更多关于不同版本的信息，你可以看看我们的文章[关于 Windows 11 的 SKU](https://www.xda-developers.com/windows-11-skus/)。除了家庭版和专业版，还有许多面向商业和教育用户的版本。不同种类的设备也有特定的版本。

## 你为什么想要 Windows 11 Pro？

正如我们提到的，Windows 11 Pro 包含了家庭版所没有的某些功能，但你到底缺少了什么？好吧，对于大多数用户来说，你可能对 Windows 11 Home 没什么问题。顾名思义，专业版的功能更适合专业人士和企业管理的电脑。尽管如此，还是有正当的理由想要它。下面让我们快速看一下这两者之间的区别。

BitLocker 加密是 Pro-exclusive 的功能之一，有助于保护您电脑上的数据免受外部攻击者的攻击。加密您的数据意味着只有您可以访问它，如果您担心数据隐私和安全，这一点很重要。

另一个独有的功能是支持远程桌面。这可以让你远程连接到你的电脑，并从任何地方控制它，例如，这对你留在办公室的电脑非常有用。还支持 Hyper-V 虚拟化。这允许您创建和运行虚拟机，因此您可以测试特定的环境或有风险的软件，而不会损坏您的主机 PC。

然而，值得注意的是，如果你有 Windows 11 Home，有几个应用程序提供类似的工具。VMware Workstation Player 是运行虚拟机的应用程序的一个例子，而 [TeamViewer](https://www.teamviewer.com/en/) 则适合访问远程电脑。不一定需要专业版。

Pro 的更多利基功能包括加入 Azure Active Directory 或业务域的能力。最初，Windows 11 Pro 还允许你用本地帐户设置你的设备，但现在，两个版本都需要互联网连接，你仍然需要一个微软帐户来设置 Windows 11 Pro 供个人使用。[尽管在两个版本中都有一个解决办法](https://www.xda-developers.com/windows-11-microsoft-local-account/)。

## 如何升级

如果你打算从 Windows 11 家庭版升级到专业版，以下是你需要做的事情:

*   打开设置 app，选择*激活*。

*   展开*升级您的 Windows 版本*部分，查看您的选项。如果你想购买 Windows 11 Pro 的升级，点击*打开商店*。

*   您将看到 Windows 家庭版和专业版之间的功能比较，并可以选择购买升级。由于 Windows 10 和 Windows 11 许可证是相同的，微软商店显示了这两个操作系统，但它将照样工作。
*   点击*购买*获得新的许可证。如果你有 Windows 11 家庭版，升级应该需要 99 美元。

*   如果您还没有输入密码，则需要输入您的密码并填写您的账单详情。

*   购买并应用升级后，您的电脑必须重启才能启用新功能。

### 使用现有的产品密钥

如果您已经有 Windows 11 Pro 的产品密钥，您可以使用以下步骤激活它:

*   导航到 Windows 11 设置应用中的*激活*页面，然后点击*更改产品密钥*旁边的*更改*。

*   您将被要求获得管理员权限来运行 Windows 激活工具，然后您可以输入您的 Windows 11 Pro 产品密钥。

*   验证密钥后，你需要重新启动电脑以启用专业版功能。

Windows 11 Home 和 Pro 其实是基于相同的安装文件，不用重装 Windows 什么的升级。简单的重新启动将配置您的 Windows 版本，下次使用您的电脑时就可以了。
