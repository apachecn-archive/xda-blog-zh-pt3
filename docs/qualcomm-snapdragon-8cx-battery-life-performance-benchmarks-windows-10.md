# 骁龙 8cx 性能指标评测显示了出色的性能和电池续航时间

> 原文：<https://www.xda-developers.com/qualcomm-snapdragon-8cx-battery-life-performance-benchmarks-windows-10/>

在去年 12 月的骁龙科技峰会上，高通发布了 T2 的骁龙 8cx T3，这是该公司为下一代“永远在线，永远连接”的个人电脑开发的高端移动平台。近 6 个月后，在台北国际电脑展上，高通终于准备好分享新移动平台的首个电池寿命和性能指标。在针对 Windows 10 的一组新的 PCMark 基准测试中，骁龙 8cx 拥有令人印象深刻的电池续航时间和性能数据。明年年初，随着高通最新 SoC 的发布，第一台“永远在线、永远连接的电脑”将会推出。

## 新的 PCMark10 应用和电池寿命曲线测试

在营销新笔记本电脑时，公司通常会发布难以置信的电池寿命指标，或者在特定情况下，这些指标对普通消费者来说毫无意义。越来越多的趋势是使用基于应用的基准来评估电池寿命和执行现实任务(如浏览网页、编辑文档或观看视频)时的性能。例如，戴尔使用名为 [MobileMark 2014](https://bapco.com/products/mobilemark-2014/) 的基于应用的基准测试来证明其声称，采用英特尔酷睿 i5 8265U、8GB 内存、256GB 固态硬盘和 FHD 显示屏的 [XPS 13](https://www.anrdoezrs.net/links/100122946/type/dlg/sid/UUxdaUeUpU24880/https://www.dell.com/en-us) 可持续使用长达 21 小时。

同样，高通也获得了 UL 开发的行业领先基准 PCMark 的帮助，以展示骁龙 8cx 笔记本电脑的真实应用性能和电池寿命。新的基准测试被称为 PCMark10 应用测试和 PCMark10 电池寿命概况测试。据 UL 工程总监 Jani Joki 称，测试工作如下:

*   PCMark10 应用测试
    *   这里没有模拟工作负载，应用程序测试根据机器在 Microsoft PowerPoint、Microsoft Word、Microsoft Excel 和 Edge 中处理真实任务的速度进行排名。
*   PCMark10 电池寿命曲线测试
    *   电池寿命测试让机器经历 3 种不同的测试。
        1.  应用程序电池寿命:在这个配置文件中，PCMark10 使用与常规应用程序测试相同的工作负载来测试实际电池寿命。测试持续 10 分钟，尽管机器完成测试越快，它空闲的时间就越长，它的分数就越高。因此，在这个测试中，性能是很重要的。
        2.  视频回放:播放 h.264 编码的 FHD 视频剪辑，以模拟“网飞高质量”视频。
        3.  空闲:机器空闲时，屏幕保持打开。
    *   应该注意的是，虽然基准测试没有对 Windows 调度程序或性能配置文件进行任何更改，但基准测试确实要求用户自行校准亮度和切换无线电。

*PCMark10 应用和电池寿命概况基准，以及 3DMark Night Raid 基准。资料来源:UL。*

如果你对这些基准如何工作的技术细节感兴趣，你可以在这里阅读完整的文档[。](https://s3.amazonaws.com/download-aws.futuremark.com/pcmark10-technical-guide.pdf)

UL 的新应用程序和电池寿命概况测试在两台运行 PCMark10 版本 2.0.2091 和 Microsoft Office 版本 1904 的笔记本电脑上进行，硬件规格如下:

1.  未命名的竞争对手笔记本电脑。
    *   英特尔酷睿 i5-8250U
    *   8GB 内存
    *   256GB NVMe 存储
    *   2K 分辨率显示器(2048x1080)
    *   49 瓦时电池
    *   Windows 10 操作系统版本:1809
2.  高通骁龙 8cx 参考笔记本电脑。
    1.  高通骁龙 8cx
    2.  8GB 内存
    3.  256GB NVMe 存储
    4.  FHD 分辨率显示屏(1920x1080)
    5.  49 瓦时电池
    6.  Windows 10 操作系统版本:1903

最后，UL 还对这两款笔记本电脑进行了 3DMark 的“夜间突袭”测试。Night Raid 是一个使用 DirectX 12 3D 图形 API 的相当新的图形基准，它旨在测试集成 GPU 的 PC 的性能。两台笔记本电脑都运行 3DMark 的 2.8.6526 版本。

在我们看任何数字之前，有几件事我们必须考虑。首先，骁龙 8cx 笔记本电脑是无风扇的，因此它能够在没有节流的情况下长期保持最佳性能，令人印象深刻。其次，基准测试使用的 Microsoft Office 应用程序是为 x86 架构构建的，因此它们通过基于 ARM 的骁龙 8cx 笔记本电脑上的仿真运行。在 ARM 上模拟 Windows 10 上的 x86 应用是一种权宜之计，直到开发人员为 ARM 重新编译他们的应用。无论如何，骁龙 8cx 笔记本电脑在 UL 的新基准中表现得非常好。

不过，高通处理器在电池续航时间上的优势是毋庸置疑的。早期骁龙驱动的“永远在线，永远连接”个人电脑的表现平平，导致许多出版物对它们的评价很差，但骁龙 8cx 似乎是一个发电站。这是高通制作的 YouTube 播放列表，展示了骁龙 8cx 的性能。今天播放列表更新了一段视频，显示该设备即使连接了两个外部显示器，也能像冠军一样处理多任务。其他视频显示笔记本电脑执行网页浏览、运行 Microsoft Excel、运行 Microsoft Outlook、运行 Microsoft Paint 3D、运行 Adobe Photoshop 和压缩文件。
