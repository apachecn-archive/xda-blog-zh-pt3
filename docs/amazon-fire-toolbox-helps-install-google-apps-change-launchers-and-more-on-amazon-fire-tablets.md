# 亚马逊 Fire Toolbox 帮助在亚马逊 Fire 平板电脑上安装谷歌应用程序、更改启动器等

> 原文：<https://www.xda-developers.com/amazon-fire-toolbox-helps-install-google-apps-change-launchers-and-more-on-amazon-fire-tablets/>

亚马逊的 Fire 平板电脑可能不会给你真正的 Android 体验，但它们足以满足用户的基本需求，尤其是孩子。他们极低的价格是以锁定的引导加载程序、没有 Google Play 认证和大量定制的皮肤(亚马逊的 Fire OS) [为代价的，这些皮肤通常基于旧的 Android 版本](https://www.xda-developers.com/amazon-updates-its-2017-fire-tablets-to-fireos-6-based-on-android-7-1-nougat/)。幸运的是，售后市场开发团体已经设法克服了这些限制。在 XDA 论坛上有很多指导，教你如何下载谷歌应用程序，替换股票启动器，以及卸载 Fire 操作系统。现在，由于 XDA 高级会员[的数据流 33](https://forum.xda-developers.com/member.php?u=8560782) ，亚马逊 Fire/Fire HD 平板电脑的用户可以更容易地下载谷歌 Play 商店，替换股票启动器，并用一个易于使用的应用程序删除所有亚马逊臃肿软件应用程序。

该应用被称为**亚马逊 Fire Toolbox** ，兼容无数 Fire 平板电脑变体，从最新版本的 Fire HD 8 到自 2014 年 Fire HD 6 以来的所有其他变体。它使用 Android 调试桥(ADB)与你的 Fire 平板电脑通信，这意味着你必须事先[启用 USB 调试](https://developer.amazon.com/docs/fire-tablets/connecting-adb-to-device.html)。一旦你安装了所需的驱动程序，应用程序可以识别你的设备，并显示如下菜单。

这些选项是不言自明的，例如，点击“谷歌助手”将帮助你用谷歌助手替换 Alexa。展开下面的列表以查看工具箱的完整功能集。

*   管理亚马逊的一切-禁用/隐藏亚马逊在平板电脑上安装的每个应用程序。
*   移除锁屏广告-自动安装 Automate 和亚马逊锁屏广告拦截器 4.5。
*   安装 Google 服务-安装 Google 框架和 Play 商店。
*   电源选项-允许您强制关闭平板电脑电源、重启、引导至恢复模式以及引导至下载模式。
*   安装 YouTube advanced-安装 YouTube 的修改版本，包括后台播放、广告拦截、强制解析等。
*   安装智能 Youtube 电视-安装智能电视和游戏机上的 Youtube 客户端。它支持 4K 视频，不需要谷歌服务，广告被屏蔽，并且在设计时考虑到了大屏幕。
*   内置更新程序-一个更新程序，检查并安装工具箱的最新版本。
*   WallChanger -将锁屏壁纸更改为自定义图像。
*   锁屏壁纸-将锁屏壁纸设置为自定义壁纸(仅限 Gens 8+)。
*   ADB Shell -启动 ADB Shell。
*   恢复一切亚马逊-启用/取消隐藏所有亚马逊应用程序回到平板电脑上。
*   侧加载应用程序-将任何应用程序直接侧加载到您的平板电脑上。
*   家长控制隐藏-隐藏家长控制产生的所有对话框。
*   安装谷歌助手-用谷歌助手替换 Alexa。
*   驱动程序检测器-如果需要，检测并安装 Fire tablet 驱动程序。
*   密度修改器-将像素密度(DPI)更改为自定义值。
*   手动恢复/删除-允许用户手动选择要恢复/删除的应用程序。
*   系统备份/恢复-创建系统备份，甚至可以恢复它们！
*   屏幕记录器-记录平板电脑的屏幕，不使用第三方工具(不记录声音或麦克风)。
*   屏幕截图——对平板电脑屏幕上的所有内容进行截图。
*   自定义启动器-允许安装自定义启动器。
*   TBShell -工具箱外壳，允许在不改变工具箱本身设置的情况下使用修改。
*   混合应用——允许用户安装网飞和迪士尼+,而无需使用亚马逊商店。
*   修改系统设置-更改导航栏的状态，禁用自动系统/应用程序更新，并关闭无线更新
*   用户管理-添加和删除用户配置文件。
*   推拉-将文件复制到平板电脑，以及将文件从平板电脑复制到 PC。

这个只支持 Windows 的应用程序是用 Visual Basic 编写的，对 Linux 的支持也在酝酿中。开发商还计划在工具箱中包括一个股票固件闪光器和一个内置的生根模块，尽管这些变化何时推出还没有时间表。
