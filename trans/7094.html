2023 年最佳游戏 CPU

原文：https://www.xda-developers.com/best-cpu-gaming/#0001-01-01

我们已经全面收集了市场上的最佳 CPU，详细介绍了不同类别和工作负载的各种 CPU 选项。然而，游戏本身值得一个专门的列表，因为选项的数量太多，不可能包含在一篇详细介绍多种工作负载的文章中。对于游戏玩家来说，现在的事情比以往任何时候都更加令人兴奋，AMD 的锐龙 7000 系列和英特尔的第 13 代 CPU都有一些非常强劲的竞争者。

游戏是建造个人电脑最受欢迎的原因之一，但获得最好的并不意味着获得你能找到的最昂贵、最符合规格的部件。游戏不需要其他任务所需要的原始马力，因此中端 CPU 几乎总是更好的价值投资。但随着游戏之外的内容创作(如流媒体和创建 YouTube 视频)的兴起，总会有你需要更多的时候。幸运的是，两家供应商都有很多不错的选择。

最佳游戏用英特尔 CPU:英特尔酷睿 i5-13600K

如果您正在寻找具有核心繁重工作负载潜力的出色游戏体验，那么酷睿 i5-13600K 就是您的理想之选。这是英特尔对 Core i5-12600K 的替代产品，此前 Core i5-12600k 占据了这一位置，尽管新的 13600K 售价 320 美元，贵了 70 美元，但它的性能足以弥补这一点。

13600K 本质上是一个 12600K，带有四个额外的 E 内核，总共有 6 个 P 内核和 8 个 E 内核，这使得 13600K 与笔记本电脑的高端 Alder Lake-P CPU 相同，如 Core i9-12900H。此外，13600K 还通过 5.1GHz 的睿频加速超越了 5GHz 大关，成为英特尔首款开箱即用的中端 CPU。但对于游戏来说，13600K 最重要的改进是其高速缓存，L2 和 L3 高速缓存的组合从 29.5MB 增加到 44MB。

与它的主要竞争对手锐龙 5 7600X 相比，13600K 交易失败。据 Techspot 报道，13600K 由于其高核心数而在生产力方面遥遥领先，但在游戏中速度稍慢，功耗也高得多。如果 13600K 的价格比游戏快一点就好了，但除非你的目标是在最苛刻的游戏中达到 240 FPS，否则你应该对这种 CPU 提供的性能水平感到满意。

如果你不确定 13600K 的价格，你应该选择 Core i5-13600KF，它没有集成显卡，但稍微便宜 310 美元。旧的 12600K 也仍然是一个选择，尽管我们不确定它会在 250 美元保持多久，因为英特尔已经提高了第 12 代部件的价格。还有 Core i5-13400，它基本上是一个 12600K，没有超频支持和较低的时钟速度，但具有第 13 代 CPU 提供的更大的高速缓存。

英特尔酷睿 i5-13600K

英特尔最新的中端 CPU 是绝对的赢家，其性能比其前代产品有很大提高，这使得它现在成为游戏 PC 中的最佳 CPU。

最佳游戏用 AMD 处理器:AMD 锐龙 5 7600X

曾经有一段时间，购买 CPU 时，你会选择 AMD 的多核多线程性能，英特尔的单核爆发。后者帮助英特尔在纯游戏领域脱颖而出，但在其他所有领域都落后于锐龙。不过，现在局面已经完全扭转了。英特尔的混合设计 CPU 具有高内核数量，而 AMD 则保留了更传统的设计，并随之带来了令人印象深刻的单核性能。

锐龙 5 7600X 现在也是 AMD 的入门级型号之一，至少目前是这样。目前没有锐龙 3，对于游戏玩家来说，锐龙 7 或锐龙 9 是多余的。甜区是最实惠的。是这个。理论上，这是熟悉的领域。6 核，12 线程，现在才第一次，还集成了显卡。基频为 4.7 GHz，提升频率为 5.3 GHz，像所有锐龙芯片一样，如果你想修补，它也是解锁的。不是说你真的需要。

正如你所希望的，在游戏方面，它超过了英特尔的酷睿 i5-13600K，这主要归功于它更大的三级高速缓存。随着 7600X 的 6 个内核赶不上 13600K 的 14 个内核，多核应用中的情况明显向 13600K 倾斜。然而，重要的是要记住，13600K 的零售价大约要高出 80 美元，所以这并不意味着 7600X 只是一个更差的 CPU。如果你主要关注游戏，7600X 仍然是一个很好的选择，在生产力方面，7600X 仍然很快，只是没有 13600K 快。

AMD 的 7000 系列是多年来第一款新的插座，正在向 AM5 转移。有了它，你也只能使用 DDR5 RAM，这在释放强大性能的同时，也增加了你的总体预算。它还支持 PCIe 5.0，这是目前有限的，但将开始在不久的将来的事情。AMD 有长期支持其平台的历史，因此即使它现在需要一个全新的系统，您也将在未来许多年内做好准备。

AMD 锐龙 5 7600X

AMD 的新入门级芯片实际上是其最适合游戏的芯片，具有令人难以置信的单核性能和诱人的价格。

最佳游戏用高性能 CPU:英特尔酷睿 i7-13700K

因此，酷睿 i5-13600K 对于游戏来说非常快，但对于游戏来说它不是最快的英特尔 CPU，并且它对于其他任务来说也缺乏原始能力。多花 100 美元，你就可以升级到酷睿 i7-13700K，它拥有两个以上的内核、更多的高速缓存和更高的时钟速度，所有这些都有助于提高性能，其中后两个对游戏尤为重要。

13700K 使用英特尔最新的 Raptor Lake 架构，该架构主要是第 12 代 Alder Lake 的更新，但增加了一些重要内容。首先，Raptor Lake CPUs 的 L2 缓存比 Alder Lake CPUs 多得多，13700K 的缓存为 24MB，几乎是旧旗舰产品 Core i9-12900K 的两倍。大多数第 13 代 CPU 也比上一代 CPU 具有更高的时钟速度，13700K 实际上在 5.4GHz 的频率上优于 12900K 的 5.4GHz。

也许英特尔对 Raptor Lake 所做的最有趣的事情是在整个堆栈中添加 E-core。这对 13700K 意味着，它实际上具有与 12900K 相同的八个 P-core 和八个 E-core 配置，但具有 Raptor Lake 体系结构的所有优势。基本上，13700K 是价格更低的更好的 12900K。酷睿 i9-13900K 拥有更多的内核(8 个 P 加 16 个 E)，但由于其更高的价格和更高的功耗，13700K 更容易推荐，而且在游戏中几乎不会慢。

Core i7-13700K 的价格约为 420 美元，相当昂贵，但客观上，对于希望以高帧率玩游戏并尽快完成工作的人来说，这是一个巨大的价值。还有 Core i7-13700KF，这是相同的 CPU，但没有集成显卡，售价略低于 400 美元。

英特尔酷睿 i7-13700K

英特尔酷睿 i7-13700K 适合那些需要比英特尔酷睿 i5-13600K 更强大功能的人。

最适合飘带:AMD 锐龙 9 7950X

AMD 最新、最强大的锐龙 CPU 将会让人们竖起耳朵。自首次上市以来，锐龙的优势之一就是难以置信的多线程性能，这对繁重的工作负载非常重要。英特尔通过其混合设计走上了一条不同的道路，但 AMD 仍然保持它更传统的一点，而锐龙 9 7950X 是 16 核和 32 线程的原始，巨大的能力。

游戏玩家可以更好地花钱，因为 16 核是多余的，但 7950X 是那些想要游戏和创造的人。当你在这样的背景下考虑时，它就更有意义了。想游戏和流？渲染高分辨率视频？用 Blender 或者虚幻引擎之类的软件？在所有这些征税过程中，有时没有什么可以替代更多。更多内核、更多线程、更高性能。

锐龙 9 7950X 是 AMD 的第一批 Zen 4，AM5 芯片之一，这是几年来我们第一次有新的插座。因此，您需要升级钻机中的所有设备。它只支持 DDR5，你也需要一个新的主板。它是为支持 PCIe 5.0 而构建的，尽管目前只有有限的使用案例，但您可以在现有的 PCIe 4.0 驱动器上转移，并在未来升级到 PCIe 5.0 设备。

在我们的测试中，锐龙 9 7950X 是你能买到的最快的 CPU 之一。总体来说，它可能比英特尔的酷睿 i9-13900K 稍慢，但一般来说，你会发现 7950X 与 13900K 相当。温度可能会稍微偏暖，但不会超过 AMD 之前宣布的 95C (203F)。在空闲时，它会很高兴地坐在 40 摄氏度(104 华氏度)的 AIO 冷却器下，你可以通过一些严重的液体冷却进一步降低温度。

最终，AMD 对锐龙家族进行了相当有意义的更新，其旗舰机型将会受到创作者的欢迎。例如，由于这种进步，需要两台电脑进行传输的日子越来越像是一去不复返了。是的，它很贵，但是它的功能是你现在在消费平台上能得到的最强大的。AM5 将在未来许多年内继续存在，因此您现在就能适应未来。

AMD 锐龙 9 7950X

AMD 最新的旗舰产品是一款完美的 CPU，适合需要大量内核来处理密集型工作负载的创作者。对游戏来说也不算太糟。

游戏预算最佳 CPU:英特尔酷睿 i5-12400

英特尔的酷睿 i5-13600K 是我们的首选，但在预算方面还有一款值得关注:英特尔酷睿 i5-12400。鉴于其出色的性能价格比，如果您正在进行预算，它值得我们大力推荐。

在规格方面，英特尔酷睿 i5-12400 的基本频率为 2.5GHz，最大睿频加速频率为 4.4GHz。我们正在考虑该芯片的六个内核和十二个线程以及 18MB 三级高速缓存。与新的 Raptor Lake 和高端的 Alder Lake 部件不同，这款芯片只有 6 个性能核心，没有效率核心。缺乏效率内核意味着它不是一个混合芯片，但它仍然在其性能内核中封装了 Golden Cove 架构。这非常适合游戏玩家，因为这意味着您仍然可以获得出色的单核性能。

至于性能，它与市场上许多其他高端游戏芯片不相上下。它提供了出色的单核性能和相对值得称赞的多线程性能，以提供令人印象深刻的游戏效果。它轻松击败了许多前代芯片，包括第 11 代的英雄产品 Core i9-11900K。11900K 在高度多核集中的任务中表现更好，但预算奥尔德湖部分在其他测试中轻松击败它。

酷睿 i5-12400 在功耗和散热性能方面也非常宽容。它在功耗方面很容易，因此产生的热量少得多。即使是一个像样的空气冷却器也足以驯服这个芯片。事实上，英特尔自己捆绑在盒子里的层状 CPU 冷却器应该足以控制该芯片的热量。这使得它成为精打细算的购物者的可靠选择，他们希望在新的建筑上尽可能地省钱。

我们建议将 Core i5-12400 与支持 DDR4 的 B660 主板配对，以获得最大的性价比。你还可以利用今天所有可用的快速 PCIe 4.0 固态硬盘，所以这是一个伟大的选择，即使是预算购物者。Core i5-12400 不是“K”系列芯片，但由于支持基准时钟超频(BCLK OC)，您仍然可以超频。也就是说，12400 在股票设置上表现很好，我们不建议超频。但是如果你喜欢修补，它就在那里。

虽然更快的 Core i5-13400 在技术上是 12400 的直接替代品，但在现实中，12400 仍然是一个很好的预算 CPU，因为它的价格比 13400 低得多。直到 12400 从市场上消失或开始以不合理的价格零售，它可能仍然是你可以买到的最好的预算 CPU。

英特尔酷睿 i5-12400 处理器

英特尔酷睿 i5-12400 是预算领域的可靠入门级产品。对于 2022 年的预算游戏来说，这是一款非常好的处理器。

最适合不带显卡的游戏:AMD 锐龙 5600G

AMD 锐龙 APU 具有集成图形处理器，并利用系统内存来提供令人印象深刻的游戏性能。AMD 在其 5000 系列下有一个坚实的 APU 阵容。锐龙 7 5700G 是一个稍微更强大的辅助动力装置，但它是锐龙 5 5600G 应该在你的电脑中占有一席之地。一旦你买了锐龙 5 5600G，围绕锐龙 7 5700G 的宣传很快就会消失，因为它能以更低的价格提供大约 96%的性能。

锐龙 5 5600G 拥有 6 个内核和 12 个线程，它是 AMD 首款 7 纳米“塞尚”桌面电脑辅助处理器的一部分。塞尚 APU 配备了 Zen 3 执行核心，搭配镭龙织女星图形引擎。锐龙 5 5600G 配备了 3.7 GHz 的基本时钟和 4.4 GHz 的增强时钟，16MB 的三级高速缓存，以及 7 个工作在 1.9GHz 的镭龙 RX Vega CUs

我们还在考虑从 45W 到 65W 的可配置 TDP，尽管您可以预期它在负载下主要在 65W TDP 下工作。作为 Zen 3 处理器，锐龙 5 5600G 从 DDR4-2933 升级到 DDR4-3200，这对 iGPU 的良好性能非常重要。顺便说一下，确保你把 5600G 和两个 RAM 配对，这样你就可以利用双通道内存。在单通道模式下运行 RAM 不仅对 CPU 性能不好，对 iGPU 性能也很糟糕。

除了 5700G 之外，锐龙 5 5600G 在集成图形引擎处理器方面是当前的领导者，尤其是在其价格点上。它能够运行市场上的大多数新游戏，尽管当谈到整体图形保真度时，你可能不得不保持你的期望。对于严肃游戏，我们推荐独立 GPU 而不是 APU，但入门级游戏玩家会发现 APU 有很多价值。它也是小型电脑或家庭娱乐电脑的完美 CPU 选择。

5600G 唯一的弱点是它是一个 AM4 CPU，AM4 平台不再有任何新的 CPU 或芯片组。更糟糕的是，唯一可以升级的 APU 是 5700G，它在游戏中并没有更好，在高度多线程的应用中速度更快。然而，如果你有一个独立的 GPU，这就打开了你的选择，你可以得到一个锐龙 9 5900X 或 5950X 的高端计算，或锐龙 7 5800X3D 的高端游戏。

AMD 锐龙 5 5600G

最佳游戏 CPU:最终想法

CPU 是你可能不会经常升级的核心组件之一，除非你口袋里有很多钱。你的 GPU 将在决定你能不能在你的电脑上玩游戏方面发挥巨大的作用，但是 CPU 也同样重要。更快的 CPU 可以让你更容易、更稳定地达到更高的帧率，尤其是在最近要求更高的游戏中。这一代，AMD 和英特尔在游戏方面都相当出色，尽管当 AMD 的锐龙 7000 3D CPUs 在 2 月份发布时，这种情况可能会发生变化。
