一个UI 5.1将为三星Galaxy S22、S21、S20以及最新一代和上一代可折叠手机提供

原文：https://www.xda-developers.com/samsung-galaxy-s22-s21-s20-z-fold-flip-4-3-one-ui-5-1/#0001-01-01

三星已经在许多手机和平板电脑上推出了其名为One UI 5.0的 Android 13 更新，但韩国OEM厂商对 Galaxy S23系列进行了小幅修改。一个UI 5.1更新是暗示将出现在精选手机和平板电脑上，但该公司没有指定任何官方路线图。尽管如此，随着Galaxy S22、S21、S20以及foldables推出新版本，推出工作已经全面展开。

银河S22

根据用户报告，Galaxy S22的稳定One UI 5.1更新已经在欧洲和美国部分地区上线。更新重量接近2GB，除了一系列新功能，它还带来了最新的2023年2月安全补丁。

对于Exynos驱动的变体，新固件以软件版本 S90xBXXU3CWAI 的形式出现。如果是美国运营商型号，制造号应为 S90xUSQU2CWAI 。

XDA论坛:银河S22 || 银河S22+ || 银河S22超

银河S21和银河S20

Galaxy S21系列的One UI 5.1更新目前正在以软件版本 G99xBXXU6EWAF 的形式向多个欧洲国家的用户推出。新版本是为手机的Exynos变种设计的，它将安全补丁级别提升到2023年2月。

XDA论坛:银河S21 || 银河S21+ || 银河S21超

另一方面，Galaxy S20已经通过build G98xxXXUFHWAK 获得了更新。从欧洲开始，Exynos车型再次领先。2023年2月的补丁紧随其后。

XDA论坛:银河S20系列

Galaxy Z Fold 4和Galaxy Z Flip 4

除了传统旗舰产品，三星还将为Galaxy Z Fold 4和Galaxy Z Flip 4推出一款UI 5.1。不过，似乎固件有两个版本。当欧洲用户正在接收f 936 bx Xu 1 WAC(Fold 4)/f 721 bx Xu 2 WAC(Flip 4)时，印度和泰国用户迎来了稍微更新的 CWB2 版本。这两种变体都带有最新的2023年2月安全补丁。

XDA论坛:Galaxy Z Fold 4| |Galaxy Z Flip 4

Galaxy Z Fold 3和Galaxy Z Flip 3

最后但同样重要的是，第三代Galaxy foldables也得到了一些UI 5.1的喜爱。目前，Fold 3的更新 F926BTBU3EWB1 和Flip 3的更新 F711BTBU4EWB1 仅在印度和泰国提供。2023年2月的补丁也包括在内。

XDA论坛:Galaxy Z Fold 3| |Galaxy Z Flip 3T21】

One UI 5.1的新增功能

安装新的更新后，用户可以期待Galaxy S23系列中许多有用的软件功能。要了解更多信息，请查看下面完整的changelog:

One UI 5.1更新

One UI 5.1通过新的图库功能以及生产力和个性化增强将您的手机提升到一个新的水平。

  • 相机和画廊
    • 快速改变自拍的色调
      • 使用屏幕顶部的效果按钮可以更容易地更改自拍的色调。

    • 发现专家原始数据
      • Expert RAW可以让您拍摄高质量的照片，非常适合那些希望完全控制照片并在以后编辑照片的人。Expert RAW现在更容易从相机的“更多”菜单中访问。

    • 更强大的搜索
      • 现在，您可以同时在图库中搜索多个人物或主题。你甚至可以搜索人，而不用标记他们的名字，只需点击他们的脸。

    • 增强的图像重制
      • 通过移除阴影和倒影，重新制作母版可以让您的照片看起来更棒。您也可以重新制作gif，以获得更好的分辨率和清晰度。预览也得到改进，以便更容易地将原始图片与重新制作的图片进行比较。

    • 创建共享的家庭相册
      • 与家人分享照片比以往任何时候都容易。通过识别家庭成员的面孔，图库将推荐图片添加到您的共享家庭相册中。每位家庭成员(最多6人)可获得5 GB存储空间。

    • 改进的信息显示
      • 当您在画廊中查看图片或视频时向上滑动，您可以看到照片拍摄的时间和地点、照片拍摄的设备、照片存储的位置等等。现在有了更简单的布局。

  • 多任务处理
    • 轻松最小化或切换到全屏
      • 现在，您可以最小化或最大化应用程序窗口，而无需进入选项菜单。只需拖动其中一个角。

    • 在分屏中访问您最常用的应用程序
      • 当您启动分屏视图时，您最常用的应用程序将显示在您最近使用的应用程序下方，以帮助您更快地找到您需要的应用程序。

    • DeX中改进的多任务处理
      • 在分屏视图中，您现在可以拖动屏幕中央的分隔线来调整两个窗口的大小。您也可以将窗口吸附到其中一个角上，使其填满四分之一的屏幕。

  • 模式和例程
    • 根据您的模式更改壁纸
      • 根据您当前的活动设置不同的壁纸。选择一种壁纸用于工作，一种用于锻炼，等等。

    • 例程的更多操作
      • 新的操作让您可以控制快速共享和触摸敏感度，更改您的铃声，以及更改您的字体样式。

  • 天气
    • 快速获取有用信息
      • 查看天气应用顶部的恶劣天气警报、每日天气摘要和日出/日落时间。温度图表现在使用颜色来显示温度在一天中的变化。

    • 每小时降水量图
      • 现在每小时的图表显示了一天中不同时间的降雨量。

    • 天气微件摘要
      • 当前天气状况的简要总结现在出现在天气窗口小部件上，让您知道是晴天、多云、下雨还是下雪。

  • 三星互联网
    • 继续在另一台设备上浏览
      • 如果您在一部Galaxy手机或平板电脑上浏览网页，然后在登录到同一Samsung帐户的另一台Galaxy设备上打开互联网应用程序，将会出现一个按钮，让您打开您在另一台设备上查看的最后一个网页。

    • 改进的搜索
      • 您的搜索现在包括书签文件夹和选项卡组的名称。改进的搜索逻辑可以让你找到你要找的东西，即使有些东西拼写不正确。

  • 其他变更
    • 让Bixby接你的电话
      • 使用Bixby text call自动接听来电，并找出对方来电的原因。您将看到来电者在文本聊天中说了什么，并且您可以轻按或键入将大声朗读给来电者的回复。Bixby文本通话仅支持英语和韩语。

    • 检查设备的电池电量
      • 新的电池插件让你可以直接在主屏幕上检查设备的电池电量。您可以查看手机、Galaxy Buds、Galaxy Watch、S Pen和其他支持的设备的剩余电池电量。

    • 在AR表情相机中最多使用3个表情
      • 在蒙版模式下与朋友一起拍摄有趣的照片和视频。你可以给每个人的脸分配不同的表情符号。

    • 设置建议
      • 登录Samsung帐户后，设置屏幕顶部会出现一些建议，帮助您在Galaxy设备上共享、连接和增强体验。

    • Spotify建议
      • 智能建议小部件现在根据您当前的活动推荐Spotify曲目和播放列表。获得驾驶、锻炼等方面的完美音乐。要获得建议，你需要在最新版本的Spotify应用程序中登录Spotify帐户。

    • 选择保存屏幕截图和屏幕录像的位置
      • 您现在可以更改保存屏幕截图和屏幕录像的文件夹。

值得注意的是，三星决定为大多数设备提供一个新的bootloader和One UI 5.1更新。因此，在你的手机上安装了这个版本之后，就没有办法降级到旧版本了。

与往常一样，上述设备的更新是成批推出的，因此您可能需要等待几天才能在您的设备上弹出更新通知。

如果你还没有在手机上收到OTA提示，你可以前往设置应用程序中的软件更新部分，检查更新是否适用于你的设备。你也可以直接从该公司的固件更新服务器(FUS)获取稳定的One UI 5.1版本，并在你的Galaxy设备上手动刷新。
