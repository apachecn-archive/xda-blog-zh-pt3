[更新:美国推出时间表]三星公布了其设备的One UI 4.0升级推出时间表

原文：https://www.xda-developers.com/samsung-revealed-one-ui-4-0-upgrade-schedule/#0001-01-01

更新(11/17/2021@ 01:56ET):三星正式分享了One UI 4.0在美国的推出时间表。滚动到底部了解更多信息。下面保留了2021年11月15日发表的文章。

在过去的几周里，三星向其相当多的设备发布了基于Android 12的One UI 4.0 beta更新。旗舰产品 Galaxy S21系列已经获得了一个UI 4.0的四个封闭测试版，而像 Galaxy S20系列和 Galaxy Note 20系列这样的老型号刚刚加入测试计划。该公司最新的可折叠产品 Galaxy Z Fold 3 和 Galaxy Z Flip 3 也已经收到了三星最新软件皮肤的测试版，三星会员应用程序上的一则新通知表明，稳定版可能很快就会开始推出。
